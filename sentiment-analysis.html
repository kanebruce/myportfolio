Media Sentiment Intelligence Dashboard

An interactive Streamlit-powered Natural Language Processing (NLP) application that analyzes news sentiment surrounding the US–Israel–Iran conflict. The dashboard transforms unstructured news articles into actionable insights through sentiment analysis, interactive visualizations, and data storytelling.

GitHub Repository Live Demo

Project Overview

The Media Sentiment Intelligence Dashboard is an end-to-end Streamlit application developed to analyze public sentiment from online news articles. It preprocesses textual data, applies Natural Language Processing techniques, classifies sentiment, and presents the results through an intuitive interactive dashboard. The application enables users to explore media narratives, understand sentiment distribution, monitor news trends, and gain meaningful insights using modern analytics techniques.

Project Impact

This project demonstrates the practical application of Natural Language Processing and data analytics for media intelligence. By transforming thousands of unstructured news articles into meaningful insights, the dashboard enables users to understand public sentiment, identify emerging trends, and make informed decisions based on data-driven evidence. The application showcases expertise in Python, Streamlit, NLP, interactive dashboards, data visualization, and analytical storytelling, making it a strong portfolio project for Data Science, Machine Learning, Business Intelligence, and Analytics roles.
